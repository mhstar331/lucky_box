# 📦 Lucky Box (Lucky Box Plugin)

인게임 인벤토리 GUI를 통해 당첨 아이템을 직접 설정하고, 전용 아이템을 사용하여 뽑기를 진행하는 마인크래프트 럭키박스 플러그인입니다.

### 🛠️ 지원 버전
- **Minecraft Version:** `1.21.11` (Paper 기반)

---

### 🎮 주요 사용법

이 플러그인은 별도의 비용 설정 명령어 대신, **전용 럭키박스 아이템**을 소모하는 직관적인 방식을 사용합니다.

#### **1. 당첨 아이템 설정 (`/lb config`)**
- 관리자 권한으로 명령어를 입력하여 54칸의 설정창을 엽니다.
- 인벤토리에 넣은 아이템들이 럭키박스에서 나올 후보가 됩니다.
- 창을 닫으면 실시간으로 자동 저장됩니다.

#### **2. 럭키박스 아이템 획득 및 사용**
- 관리자는 `/lb item` 명령어로 특수 모델 데이터(`CustomModelData: 1`)가 적용된 종이 아이템을 지급받을 수 있습니다.
- 유저는 이 아이템을 손에 들고 **우클릭**하면 아이템 1개가 소모되며 즉시 뽑기 애니메이션이 시작됩니다.

---

### 📜 주요 명령어
모든 명령어는 `/lucky_box`, `/luckybox`, `/lb`, `/럭키박스`로 사용 가능합니다.

| 명령어          | 설명                            | 권한               |
|:-------------|:------------------------------|:-----------------|
| `/lb item`   | 럭키박스 전용 아이템(종이)을 1개 지급받습니다.   | `luckybox.admin` |
| `/lb config` | 당첨 아이템 목록을 설정하는 54칸 GUI를 엽니다. | `luckybox.admin` |

---

### 🔑 권한 (Permissions)
[LuckPerms 플러그인](https://modrinth.com/plugin/luckperms) 등을 통해 유저별 권한을 관리할 수 있습니다.

- **`luckybox.use`**: 럭키박스 아이템을 우클릭하여 사용할 수 있는 권한 (기본 유저용)
- **`luckybox.admin`**: 당첨 아이템 설정 및 전용 아이템 지급 권한 (관리자용)

---

### ✨ 주요 특징
- **다국어 자동 지원**: 당첨 시 출력되는 아이템 이름은 접속한 유저의 마인크래프트 언어 설정(한국어, 영어 등)에 맞춰 자동으로 번역되어 표시됩니다.
- **Ease-out 애니메이션**: 뽑기 진행 시 슬롯이 돌아가는 속도가 점점 느려지며 결과가 공개되어 몰입감을 줍니다.
- **리소스팩 연동**: 전용 리소스팩이 포함되어 있어, 기본 종이 아이템이 입체적인 상자 모양으로 보이도록 설정되어 있습니다. (서버 접속 시 자동 적용 기능 포함)

---

### ⚠️ 주의 사항
- **아이템 소모**: 우클릭 시 인벤토리에서 아이템 수량이 즉시 1개 차감됩니다.
- **데이터 저장**: 아이템 정보는 `config.yml`에 저장되므로, 파일을 직접 수정하기보다 인게임 GUI 사용을 권장합니다.
